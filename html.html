HTML

Syntax

HTML tags define elements that surround markup text and can have attributes.

  • Tag: A region of text formatted with angle brackets < like so >
  • Opening Tag: The start of an HTML element
  • Closing Tag: The end of an HTML element - shorthand is </Tag>
  • Element: A node within an HTML document, denoted by HTML opening and closing tags
  • Markup Text: Either plain text or additional HTML markup
  • Attributes: Key/Value pairs that supply the HTML engine with appropriate context for an element

    • Page Tags

    These tags help define the shape of your HTML document. You'll use them to denote major regions of the file.

    Tag Description
    <html> Opens an HTML document
    <head> Opens an HTML heading from within a <html> block
    <style> Opens a CSS block (typically used within an HTML heading)
    <script> Allows you to link or embed scripts within an HTML document
    <link> Allows you to link CSS media to your HTML page
    <body> The main content area of an HTML page
    <h1>
    <h2>
    <h3>
    <h4>
    <h5>
    <h6>     		Headings (Level 1 through 6)

    Table Tags

    These tags are used to format tables.

    Tag Description
    <table> Creates a new HTML table
    <tr> Table Row
    <th> Table Heading
    <td> Table Cell (Data)

    Content Tags

    These tags give you some basic text formatting capabilities. Note that modern HTML prefers CSS over tags for formatting blocks of text.

    Tag Description
    <p> Paragraph block
    <br/> Line Break
    <img> Image tag - embeds a .png, .jpg, .gif, etc. inline with your text
    Note: You can also place images as backgrounds via CSS.
    <!-- --> Comment - not visible on the page, but available in your editor
    <hr> Horizontal Rule - separate a region of text
    <div> Division of text - used to block text that should produce a content break afterwards
    <span> Formats text inline, typically within a span of another content block.
    <a> Anchor Tag (Link)
    <em> Emphasized point within a block of text
    <ul> or <ol> Unordered and Ordered Lists
    <b> or <strong> BOLD Text
    <i> Italic Text
    <u> Underlined Text
